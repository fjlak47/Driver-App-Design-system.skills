# 滴滴货运司机端设计规范

## 版本信息
- **当前版本**: v1.2.0
- **更新日期**: 2026-04-10

## 项目概述
本项目包含滴滴货运司机端的设计规范，包括字体、颜色、组件等设计元素的标准化定义。

## 主要文件
- `滴滴货运司机端设计规范.skills` - 设计规范主文件（JSON格式）
- `SKILL.md` - 设计规范技能文档
- `design-spec-visual.html` - 设计规范可视化页面
- `references/design-spec.md` - 设计规范摘要

## 设计规范要点
- **设计画布宽度**: 375px
- **字体**: PingFang SC（英文与数字优先 Barlow）
- **主色调**: #FF6435
- **卡片间距**: 10px
- **滚动条**: 隐藏

## 尺寸换算规则
- 所有来自 MasterGo 的设计稿均基于 750px 宽度画布
- 本设计规范基于 375px 宽度画布
- 从 MasterGo 读取的尺寸数值，需自动除以 2 后再写入本规范

## 如何使用
1. 查看 `design-spec-visual.html` 了解设计规范的可视化展示
2. 参考 `SKILL.md` 和 `references/design-spec.md` 获取详细规范信息
3. 使用 `滴滴货运司机端设计规范.skills` 文件作为设计工具的参考
